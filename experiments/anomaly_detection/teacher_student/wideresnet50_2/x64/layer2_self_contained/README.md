# TS-WideResNet50-2 Layer2 Self-Contained

This folder contains the self-contained single-layer WideResNet50-2 teacher-student notebook and configs.

## What This Branch Is For

- experimenting with the WideResNet50-2 teacher-student setup end to end in one notebook
- rerunning or rebuilding the WideResNet teacher-student workflow from scratch

## Notes

This branch is the saved artifact-backed WideResNet50-2 path in the repo.

- checkpoints: `artifacts/ts_wideresnet50_layer2/checkpoints/`
- results: `artifacts/ts_wideresnet50_layer2/results/`
- plots: `artifacts/ts_wideresnet50_layer2/plots/`

For the true multilayer self-contained notebook, use the sibling `../multilayer_self_contained/` folder.
