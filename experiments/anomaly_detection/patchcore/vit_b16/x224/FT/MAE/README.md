# PatchCore ViT-B/16 MAE Fine-Tuned (`x224`)

This experiment fine-tunes the ViT-B/16 backbone on WM-811K normal wafers via a masked patch reconstruction (MAE-style) pretext task, then builds a standard PatchCore memory bank on the domain-adapted features. All PatchCore hyperparameters are held identical to the frozen ViT-B/16 main experiment to isolate the effect of the backbone weights.

## Method

**Stage 1 — Self-Supervised Fine-Tuning.**
The last 6 transformer blocks (blocks 6–11) and the final LayerNorm of a pretrained ViT-B/16 are updated via masked patch reconstruction: 75% of the 196 patch tokens are randomly masked, and the model is trained to reconstruct the masked patch pixel content from the visible context using a 2-layer MLP decoder (768 → 512 → 768). No defect labels are used. Fine-tuning runs for 10 epochs with AdamW (LR = 1×10⁻⁴) and mixed-precision (fp16).

**Stage 2 — PatchCore Memory Bank.**
Block-6 patch tokens are extracted from the fine-tuned backbone, projected to 128 dimensions, and L2-normalised. A random-sampled memory bank of 600,000 tokens is built from the 40k training normals — identical to the frozen backbone setup.

**Stage 3 — Threshold Calibration and Evaluation.**
Threshold is the 95th percentile of tune-normal scores (same rule as all other experiments). Image-level scores are the mean of the top-10% patch distances to the memory bank.

## Comparison with Frozen ViT-B/16 (controlled)

All PatchCore settings are identical. Only the backbone weights differ.

| Metric | Frozen ViT-B/16 | MAE Fine-Tuned | Delta |
|---|---|---|---|
| F1 | 0.595 | 0.595 | +0.000 |
| AUROC | 0.956 | 0.959 | +0.003 |
| AUPRC | 0.671 | 0.717 | **+0.046** |
| Recall | ~80% | 82.4% | +2.4pp |
| FPR | 5.0% | 4.74% | -0.26pp |

MAE fine-tuning does not change deployed F1 but meaningfully improves AUPRC (+0.046), indicating better precision-recall quality across all thresholds.

## Notes

- Blocks 0–5 remain frozen throughout; only blocks 6–11 and the LayerNorm are updated.
- MAE reconstruction loss (masked-patch MSE) decreases from 0.0694 (epoch 1) to 0.0420 (epoch 10), confirming stable convergence.
- AUPRC improvement (+0.046) is the primary finding. F1 at the 95th-percentile threshold is unchanged because the threshold calibration method already targets 5% FPR for both models.
- Scratch recall degrades (72.7% → 45.5%), likely because MAE reconstruction emphasises broad spatial coherence more than fine-grained local texture, which thin-streak defects require.
