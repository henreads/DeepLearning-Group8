# EfficientNet-B0 PatchCore

This branch stores the local EfficientNet-B0 PatchCore `x224` benchmark run.

Files:
- `notebook.ipynb`: canonical train-or-reuse notebook for this branch
- `artifacts/checkpoints/`: saved checkpoint bundle used by the notebook when retraining is skipped
- `artifacts/results/`: saved summaries, score CSVs, threshold sweep outputs, and defect-analysis tables
- `artifacts/plots/`: figures regenerated by the notebook from the saved checkpoint and result files
- `scripts/run_patchcore_effb0_x224_main_notebook.py`: local runner used by the notebook when `RUN_TRAINING = True`

Notes:
- The notebook is intended to run top to bottom in both modes: reuse the saved checkpoint and result files by default, or launch the local runner to regenerate the branch artifacts.
- The original archived source notebook is still preserved under `experiments/archive/anomaly_50k_duplicates/21_patchcore_efficientnet_b0_5pct.ipynb`.
