# Anomaly 50k Notebooks

This folder contains the original anomaly-detection notebook sequence that was previously stored directly under `notebooks/`.

Scope:

- dataset family: `data/processed/x64/wm811k`
- default benchmark metadata: `data/processed/x64/wm811k/metadata_50k_5pct.csv`
- notebook sequence: `01` through `20`

Why this folder exists:

- to make it explicit that these notebooks belong to the original `50k` benchmark split
- to keep them visually separate from the larger labeled workflows under `notebooks/anomaly_120k_labeled/`
- to leave room for additional dataset-specific notebook groups without overloading the top-level `notebooks/` directory
